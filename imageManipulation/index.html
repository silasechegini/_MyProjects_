About

This is a simple project that deals with image manipulation. For different browsers, there will be different requirements for different image sizes to fit into the browsers appropriately. considering large scale projects where there are a lot of images in a galery, it may be cumbersome to make changes to all the images differently. a simple script can handle this procedure in a repeatable fashion.
