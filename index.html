Solana Networks
Wireless Software Developer/Researcher; November 2018 - May 2019
Ottawa, Ontario

I joined the Wireless software research team at Solana Networks in November 2019 as a Wireless Software developer and researcher, focused on building simulators and writing algorithms for Geo-location estimation in wireless ad-hoc environments. I came into the team with knowledge of the BGP, TCP and UDP protocols, the OSI model, Network Simulators, Wireless Networks, Mobile Ad-Hoc Networks (MANETs), C++, C, Python and Git which were required for this role.
I was able to apply my software developement skills to building a mobile ad-hoc network (MANET) simulator, following the available OSI model implementation in the OmNet++ network simulator, designed and implemented geolocation estimation algorithms in C++ for geolocation triangulation using Received Signal Strength (RSS) and Time of Arrival (TOA) methods - all of these implementations were based on research papers published in this field. Python was the tool i used to automate the experiments in the network simulator and to evaluate the results obtained. For every successful implementation of both algorithms, test were carried out in real time on Mobile adhoc network test boards and the results obtained were in close range with the simulator implementation upto a 90% degree of accuracy.
